# 영어는 김현정 영어전문학원 — 웹사이트 기능 목록

> 이 문서는 이 웹사이트가 가진 기능을 **일반인이 알아보기 쉽게** 정리한 것입니다.

---

## 1. 누가 쓰나요?

- **관리자(선생님)** : 학생 등록, 영상 등록·할당, 반 관리, 진도 확인, 리포트 공유 등
- **학생** : 로그인 후 할당된 영상 시청, 진도 저장, 비밀번호·이메일 변경, 리포트 보기
- **학부모** : 학생과 같은 계정으로 "리포트"만 보거나, 관리자가 공유한 링크로 학습 현황 확인

---

## 2. 로그인·계정

| 기능 | 설명 |
|------|------|
| **관리자 로그인** | 이메일 + 비밀번호로 로그인 → 관리자 화면(대시보드)으로 이동 |
| **학생 로그인** | 등록된 **이름** + 비밀번호로 로그인 → 학생 화면으로 이동 (이메일은 관리자가 등록한 값으로 자동 처리) |
| **비밀번호 재설정** | "비밀번호를 잊으셨나요?" 클릭 → 이름 + **본인 이메일** 입력 → 해당 이메일로 재설정 링크 발송. 링크에서 새 비밀번호 설정 |
| **비밀번호 변경** | 로그인한 학생이 학생 페이지에서 새 비밀번호로 변경 가능 |
| **내 이메일 등록** | 로그인한 학생이 본인 이메일을 직접 등록·수정 (비밀번호 재설정·이메일 로그인에 사용) |

---

## 3. 관리자 — 대시보드

| 기능 | 설명 |
|------|------|
| **학생 등록** | 이름 + 비밀번호만 입력해서 새 학생 계정 생성 (이메일은 나중에 학생이 직접 등록) |
| **학생 목록** | 재원생 / 퇴원생 탭으로 구분해 목록 표시 |
| **반 지정** | 학생별로 "반" 선택 (드롭다운) |
| **영상 할당** | 특정 학생에게 영상 할당. **방식 1:** YouTube URL 입력. **방식 2:** "등록된 재생목록/동영상에서 할당" 버튼 → 재생목록·동영상 제목으로 검색 후 선택해 할당 |
| **제목으로 검색** | 등록된 재생목록·동영상 목록을 **제목으로 검색**해서 빠르게 찾기 |
| **퇴원 처리** | 재원생 한 명을 퇴원생으로 전환 (계정·진도는 유지, 목록만 퇴원생 탭으로 이동) |
| **재원 복귀** | 퇴원생을 다시 재원생으로 전환 |
| **완전 삭제** | 재원생/퇴원생 계정과 진도 기록을 완전히 삭제 (복구 불가) |
| **리포트 공유** | 학생별로 "리포트 공유" 켜기/끄기 + **공유 링크 복사**. 학부모가 이 링크로 들어가면 해당 학생의 학습 리포트만 볼 수 있음 |
| **배정된 영상 보기** | 학생별로 펼치면, 그 학생에게 할당된 영상 목록과 진도율·마지막 시청 시각 확인 |

---

## 4. 관리자 — 영상 관리

| 기능 | 설명 |
|------|------|
| **단일 영상 등록** | YouTube 영상 URL 입력 → 한 개 영상만 등록 |
| **재생목록으로 일괄 등록** | YouTube 재생목록 URL 입력 → 재생목록 제목으로 **강좌(재생목록)** 생성 후, 재생목록 안 영상들을 한 번에 등록 |
| **등록된 재생목록 / 등록된 영상** | 탭으로 구분. 재생목록별로 묶어서 보기, 또는 단독 영상만 보기 |
| **제목으로 검색** | 영상·재생목록 **제목으로 검색**해서 목록 필터링 |
| **선택 항목 학생/반에 할당** | 체크한 영상(또는 재생목록 내 영상)을 **반 전체** 또는 **선택한 학생들**에게 한 번에 할당 |
| **선택 항목 노출/주간과제 설정** | 특정 영상을 "학생에게 보이기/숨기기", "주간 과제로 지정" 등을 **학생별·반별**로 설정 |
| **영상 삭제** | 단일 삭제 또는 선택한 여러 개 일괄 삭제 (배정된 학습 기록에 영향 있음) |

---

## 5. 관리자 — 반 관리

| 기능 | 설명 |
|------|------|
| **반 추가** | 반 이름 입력해서 새 반 생성 |
| **반별 학생 수·진도** | 반 목록에서 반별 학생 수, 평균 진도 등 확인 |
| **반에 영상 일괄 배정** | 반 선택 + **등록된 재생목록/동영상**에서 영상 선택(제목 검색 가능) → 그 반 소속 학생 전원에게 선택한 영상 배정 |

---

## 6. 관리자 — 배정 목록

| 기능 | 설명 |
|------|------|
| **1:1 배정** | "영상 선택"(제목 검색 가능) + "학생 선택" → 한 학생에게 한 영상 배정 |
| **배정 목록·진도** | 학생별로 펼치면 배정된 영상 목록, 진도율, 완료 여부, 마지막 시청 시각 확인 |
| **필터** | 학생별로 "전체 / 완료 / 미완료" 필터로 보기 |
| **배정 해제** | 특정 배정 한 건 해제 (영상 삭제 아님, 해당 학생에게만 배정만 제거) |

---

## 7. 학생 — 학생 보기

| 기능 | 설명 |
|------|------|
| **할당된 영상 목록** | 관리자가 할당한 영상이 카드 형태로 표시. 클릭하면 시청 페이지로 이동 |
| **진도·완료** | 영상별 진도율, 완료 여부(주간 과제 여부 포함) 표시 |
| **내 이메일 등록/수정** | 본인 이메일 입력 후 저장 (비밀번호 재설정 등에 사용) |
| **비밀번호 변경** | 새 비밀번호 입력 후 변경 (다음 로그인부터 적용) |
| **PWA(앱처럼 사용)** | 모바일에서 "홈 화면에 추가" 안내로 앱처럼 설치·실행 가능 |

---

## 8. 학생 — 리포트(학부모 보기)

| 기능 | 설명 |
|------|------|
| **학생 보기 / 리포트 탭** | 같은 학생 계정으로 "학생 보기"(영상 목록)와 "리포트(학부모 보기)" 탭 전환 |
| **리포트 내용** | 주간·월간 학습 완료율, 최근 시청한 영상 목록과 진도·완료 여부 등 (학부모가 볼 수 있는 요약 정보) |

---

## 9. 학부모 — 리포트 링크

| 기능 | 설명 |
|------|------|
| **리포트 전용 링크** | 관리자가 "리포트 공유"로 복사한 링크(`/report/토큰`)를 열면, **로그인 없이** 해당 학생의 학습 리포트만 볼 수 있음 |
| **주간 학습 리포트** | 주간·월간 완료율, 최근 영상 진도 등 표시 |

---

## 10. 영상 시청

| 기능 | 설명 |
|------|------|
| **시청 페이지** | 학생이 할당된 영상 카드를 클릭하면 `/watch/배정ID` 로 이동. YouTube 영상이 페이지 안에 재생됨 |
| **진도 자동 저장** | 재생 위치·진도율이 주기적으로 저장되어, 다음에 이어보기 가능 |
| **완료 처리** | 영상 대부분(95% 이상) 시청 시 자동으로 "완료" 처리 |
| **건너뛰기 제한** | 아직 시청하지 않은 구간으로 앞으로 감기 시, 자동으로 마지막 시청 위치로 되돌림 |
| **배속 고정** | 배속 변경 불가. **1배속(1.0)만** 사용 가능. 변경 시도 시 "배속 변경은 허용되지 않습니다" Toast 표시 |
| **전체화면 비활성화** | 플레이어에서 전체화면 버튼 숨김 (인라인 재생만 가능) |

|------|------|

## 12. 요약

- **관리자**: 학생 등록·반 관리·영상 등록·재생목록/동영상 할당(URL 또는 등록 목록+제목 검색)·퇴원/재원/삭제·리포트 링크 공유·배정 목록·진도 확인
- **학생**: 이름+비번 로그인, 할당 영상 시청, 진도 자동 저장, 이메일·비번 변경, 학생 보기/리포트 탭
- **학부모**: 리포트 링크로 로그인 없이 학습 현황 확인
- **영상 재생**: 1배속 고정, 건너뛰기 제한, 전체화면 비활성화, 진도 자동 저장
